# Examples

- `sample_references.md`: minimal input example
- `sample_ref_manifest.csv`: minimal output example for spreadsheet review
- `sample_ref_manifest.json`: sample machine-readable output
- `sample_ref_manifest.md`: sample human-readable output with bilingual field notes for `article_url` and `pdf_url_hint`

Run:

```bash
python scripts/extract_ref_manifest.py examples/sample_references.md --out-dir examples/out
```
